.. _configuration:

Configuration
=============

The main configuration is separated in four parts:

- The output configuration (What to display and how).
- The register configuration (Special registration options like douple-opt-in).
- The edit configuration (Special edit options like send-edit-mail).
- The validation configuration (In which way the fields are validated).
- The redirect configuration (Defines the redirect page for each step).
- The label configuration (The displayed labels or what to display if an error occoured).
