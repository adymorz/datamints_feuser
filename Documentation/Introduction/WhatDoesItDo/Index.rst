What does it do?
----------------

This extension provides two main functionalities:

- Frontend user registration.
- Frontend user edit form.

It's a powerful Frontend user management which is simple to implement and provides many features:

- A complete configurable Frontend output.
- A predefined configuration that can be included as static template like the "css-styled-content"
  configuration.
- A zero configuration md5 system that can be activated with extensions like "saltedpasswords"
  (TYPO3 >= v4.3), "md5passwords" or "t3sec_saltedpw".
- A configurable validation system for every field in fe_user table.
- Sending HTML or plain text e-mails configurable by a template.
- Double-opt-in, admin approval and auto login system.
- Configurable redirect sites at each step of the registration or edit process.
- Consultancy for undocumented salesforce integration.
